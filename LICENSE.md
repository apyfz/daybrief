# License — decision pending

Daybrief intends to launch under a **protective license** (SPEC §12) to keep the
productization door open. The leading candidate is **AGPL-3.0-or-later**; a
fair-source / BSL-style license is the alternative.

This choice must be finalized **before the first external contribution** —
relicensing is hard once a contributor base forms. Until then, `package.json` and
crate metadata declare `AGPL-3.0-or-later` as the working intent.

**Action item:** replace this file with the full text of the chosen license
(e.g. the verbatim GNU AGPL v3) before accepting outside PRs.
