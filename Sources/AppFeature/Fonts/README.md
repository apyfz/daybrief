# Fonts

Daybrief uses **Tiempos Text** (Klim Type Foundry) for its editorial serif when present.

The `.ttf` files here are **git-ignored** — Tiempos is a commercial/trial font that
must not be committed to or distributed with this (AGPL) repository. Drop your own
licensed `TiemposText-Regular.ttf` and `TiemposText-RegularItalic.ttf` into this
folder for the editorial serif; without them, Daybrief falls back to the system serif.
